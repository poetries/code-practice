逻辑运算
逻辑运算 &&, ||,
在PHP中,逻辑运算返回的是布尔值, true /false
而在js中,逻辑运算稍有特殊,

返回的是能确定整体表达式的值的第一个表达的值.



// 这些东西被用来做逻辑判断的时候当成假
false, 0, '', null ,undefined,
这些是假,除外当真


var res = 3 || null;
alert(res); // 3

var res 0 || false;
alert(res); // false;

var res = true && 9;
alert(res); // 9

var res = false && true
alert(res); // false;


3元运算
表达式1 ? 表达式2 : 表达式3;
表达1为真,则执行表达式2,否则 执行表达式3

if(表达式1){
    表达式2;
} else {
    表达式3;
}
